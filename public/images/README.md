# Site images

## Default social sharing image (Open Graph)

- **File:** `og-default.jpg`
- **Recommended size:** 1200×630 px (aspect ratio 1.91:1)
- **Use:** Shown when the site is shared on social networks (Facebook, LinkedIn, Twitter, etc.) when a page doesn’t set its own image.

To use your own image:

1. Create or export an image at **1200×630 pixels**.
2. Save it as `og-default.jpg` in this folder (`public/images/`).
3. Rebuild/redeploy the site.

For best results use your logo and tagline on a brand-colored background, with minimal text so it stays readable at small sizes.
